# Raukeeyang Integrated Farm Website

This static website is ready for deployment as a GitHub Pages site.

## Files
- `index.html` — Home page
- `products.html` — Product catalogue
- `contact.html` — Contact form page
- `thankyou.html` — Submission confirmation page
- `css/styles.css` — Global styles
- `js/` — Interactive page scripts
- `images/` — WebP images and icons
- `data/products.json` — Product data used by JavaScript

## Deployment
1. Push the repository to GitHub.
2. In the repository settings, enable **GitHub Pages**.
3. Set the source branch to `main` (or `master`) and the folder to `/ (root)`.
4. Wait a few minutes for GitHub Pages to build the site.

## Notes
- All internal navigation points to `contact.html`.
- The duplicate file `contacts.html` has been removed to keep the site clean and consistent.
- The project uses optimized `.webp` images for faster page loading.
