# WDD231 Projects

This repository contains projects and assignments developed during the WDD231 course, demonstrating skills in modern web development using HTML, CSS, and JavaScript.

## Course Outcomes

- Develop dynamic websites using valid HTML and CSS while following best practices for accessibility and standards compliance.
- Create dynamic websites that leverage browser APIs, JSON, and remote APIs.
- Use industry tools to monitor performance and optimize the user experience.
- Demonstrate the traits of an effective team member, including clear communication, collaboration, completing assignments, and meeting deadlines.
